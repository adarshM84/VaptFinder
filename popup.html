About VaptFinder

VaptFinder is a lightweight, privacy-focused Chrome Extension designed to help developers and security enthusiasts quickly identify potential vulnerabilities in web technologies.

It serves two main purposes:

  • Passive Scan: Automatically scans the current tab for common JavaScript libraries (like React, jQuery, Lodash) and checks if the detected version has known vulnerabilities.
  • Product Lookup: Switch tabs to manually check any package you are interested in.

It is open source and created by Adarsh Mishra and can be found on GitHub.

For more information, visit the GitHub repository.

It is designed with privacy in mind. It does not track you, uses no analytics, and sends data only when necessary to perform a vulnerability check.
